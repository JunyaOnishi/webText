社内研修用リポジトリ
======================
これは、新人向け社内研修用のリポジトリです。
テキストに関しては、社内サーバーだけで使用します、こちらに放り込んだりあげたりしないこと。

使い方
------
- [Download ZIP](https://github.com/JunyaOnishi/webText/archive/master.zip)をクリックしてください。
- Gitの使用はGitの扱い方の学習の際に使用します。
